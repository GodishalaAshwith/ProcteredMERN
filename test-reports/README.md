This directory contains comprehensive test reports generated by the automated testing system for the Proctored MERN Exam Application.

### Backend API Test Reports

- **backend-report.html** - Detailed Jest test results with pass/fail status for each test
- **backend-coverage/** - Code coverage reports showing line, branch, and function coverage
  - `index.html` - Main coverage dashboard
  - `lcov-report/` - Detailed line-by-line coverage analysis

### Frontend E2E Test Reports

- **frontend-e2e-reports/** - Cypress end-to-end test results
  - `merged-report.html` - Consolidated test results from all spec files
  - Individual spec reports with screenshots and videos of test runs

### Comprehensive Summary

- **comprehensive-test-report-[timestamp].html** - Executive summary with all test results
  - Overall pass/fail status
  - Feature coverage matrix
  - Links to detailed reports
  - Test environment information

### Failure Artifacts (when tests fail)

- **cypress-screenshots/** - Screenshots captured during test failures
- **cypress-videos/** - Video recordings of failed test runs
- **error-logs/** - Detailed error logs and stack traces

## ðŸ”§ Automation Details

### Test Execution

- **Backend Tests**: Jest + Supertest for API testing
- **Frontend Tests**: Cypress for E2E user flow testing
- **Database**: Isolated test database with automatic cleanup
- **Environment**: Dockerized services for consistency

### CI/CD Integration

- Tests run automatically on every commit
- Pull request validation prevents broken code
- Deployment blocked if tests fail
- Reports published to GitHub Pages

### Local Development

- Run `./run-tests.sh` (Linux/Mac) or `./run-tests.ps1` (Windows)
- Individual test suites: `./run-tests.sh backend` or `./run-tests.sh frontend`
- Reports generated in this directory after each run
